# NumPy, pandas and ML Foundations

This project is a practical introduction to the core Python libraries used in data analysis and machine learning.

## Tools used

- NumPy
- pandas
- matplotlib
- seaborn
- scipy
- scikit-learn
- Jupyter Notebook

## What I practiced

- Matrix and vector operations with NumPy
- Data manipulation with pandas
- Function plotting with matplotlib
- Optimization with scipy
- Confidence interval visualization with seaborn
- kNN classification with scikit-learn
- Cross-validation and accuracy evaluation
